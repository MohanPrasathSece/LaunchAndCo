1. **Unsplash** (https://unsplash.com)

   - Best for: Professional business photos, offices, teams

2. **Pexels** (https://pexels.com)
   - Free for commercial use
   - Best for: Corporate settings, meetings, technology

3. **Pixabay** (https://pixabay.com)
   - Free for commercial use
   - Good variety of business images

## Image Optimization Tips

1. **Resize images** before uploading:
   - Use tools like TinyPNG (https://tinypng.com) to compress
   - Keep file sizes under 200KB each

2. **Format:**
   - Use JPG for photos
   - Use WebP for better compression (optional)

3. **Naming Convention:**
   - Use descriptive names with hyphens
   - Follow the naming pattern suggested above

4. **Alt Text:**
   - Each image has meaningful alt text in the code
   - Improves SEO and accessibility

---

## Quick Download Instructions

1. Go to Unsplash.com or Pexels.com
2. Search for the recommended terms
3. Download the Medium or Large size
4. Save with the exact filename suggested
5. Place in the `public/images/` folder
6. Refresh your browser - images will appear automatically!
